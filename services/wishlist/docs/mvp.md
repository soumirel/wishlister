# wishlister

Система для трекинга виш-листов

## user stories

Я, как пользователь, хочу:

- [ ] Создать учётную запись:
- [x] Авторизоваться в свою учётную запись
- [x] Добавлять пожелания в вишлист
- [x] Удалять желания из вишлиста
- [x] Просматривать свой вишлист
- [x] Просматривать чужие вишлисты
- [x] Бронировать в чужих вишлистах пожелания
    - [x] Другие пользователи не могут забронировать это пожелание
- [x] Видеть, какие желания забронированы в чужих вишлистах
