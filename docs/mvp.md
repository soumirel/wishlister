# wishlister

Система для трекинга виш-листов

## user stories

Я, как пользователь, хочу:

- Создать учётную запись:
	- Имя
- Авторизоваться в свою учётную запись
- Добавлять пожелания в вишлист
- Желания составлять из:
    - Названия
    - Описания
    - Картинки
- Удалять желания из вишлиста
- Делиться вишлистом
- Анонмно бронировать в чужих вишлистах пожелания
    - Другие пользователи не могут забронировать это пожелание
- Видеть, какие желания забронированы в чужих вишлистах
